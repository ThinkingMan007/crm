积分查询
      (1)用户输入查询条件，点击“查询”按钮，可以查询出符合条件的固定客户，以及固定客户的积分信息
      (2)查询到的固定客户会在上面的列表框中显示出来，点击固定客户，该固定客户下的所有联系人信息会显示在下面的列表框中。其中，主联系人信息背景为红色，并且排在第一行；
      (3)点击下面列表框的每个标签，可以查看该固定客户的积分来源与使用明细：
    • a.点击“联系人积分”，可查看该固定客户下联系人的积分明细；
    • b.点击“礼品申请记录”，可以查看该固定客户所有联系人积分兑换记录；
    • c.点击“已积分运单”，可以查看通过运单获得的积分信息。输入日期查询条件，可查询出指定时间段的运单积分信息；
    • e.点击“奖励积分列表”，可以查看该固定客户获得的积分奖励信息；
    • e.点击“调整积分”，可以查看该固定客户下联系人因运单变更、联系人调整和主联系人调用等原因，所产生的积分变动情况。
