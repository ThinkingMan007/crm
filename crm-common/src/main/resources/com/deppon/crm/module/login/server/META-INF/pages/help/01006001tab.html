工单上报
      (1)上报工单：
    • a.输入凭证号，凭证号可以为运单号或订单号，点击回车或者“GO”可检索该工单是否上报过工单，并显示在上方的工单列表中；
    • b.输入联系电话，可根据CC系统来电电话自动导入，可编辑，点击回车可检索该来电号码的工单记录，显示在上方工单列表中；
    • c.输入来电人，可选择发货方或收货方，同时录入来电客户称呼；
    • d.选择上报类型，可根据来电按钮，或客户请求选择投诉或异常；
    • e.选择工单来源、期望时限和优先级别；
    • f.输入上报内容和客户要求；
    • g.若是前台已处理，勾选最终反馈；
    • h.点击“提交”，完成工单上报。
      (2)重置
    • 点击“重置”，填写的上报内容全部清空。
