订单分配
      (1)查询
    • 输入查询条件，点击“查询”按钮，系统会显示符合查询条件的订单信息。
      (2)重置
  1. 若需重新更改查询条件，点击“重置”按钮，可清空原输入的查询条件。
      2. (3)分配订单
    • a.在查询结果列表中选中一条需要分配的订单;
    • b.点击“查询地图”，选择始发网点;
    • c.点击“分配订单”按钮，订单流向始发网点对应的受理部门。
      (4)拒绝订单
    • a.在查询结果列表中选中一条需要拒绝的订单，选择反馈信息（即拒绝原因）;
    • b.点击“拒绝订单”按钮，拒绝订单成功，订单状态更新为已拒绝
      (5)撤销订单
    • a.在查询结果列表中选中一条需要撤销的订单，选择反馈信息（即撤销原因）;
    • b.点击“撤销订单”按钮。撤销订单成功，订单状态更新为已撤销。
      (6)催促办理
    • a.在查询结果列表中选中一条需要催促办理的订单;
    • b.点击“催促办理”按钮。催单成功，系统会提示催办成功，此内容会在催单对象部门的待办事项中显示。
