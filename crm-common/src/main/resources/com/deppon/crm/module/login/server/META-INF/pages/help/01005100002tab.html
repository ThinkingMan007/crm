开发计划管理
      (1)查看开发计划
    • a.填写查询条件，“开发时限”的最大查询范围为三个月且必填，点击“查询”按钮，显示对应的查询结果；
    • b.选中需要查看的开发计划，点击“查看”按钮，或双击该开发计划，弹窗显示该计划详情。
      (2)新增开发计划
    • a.点击“新增”按钮，弹出新增开发计划界面；
    • b.查询客户：客户创建时间默认一个月，最大查询范围为一年；
    • c.选择客户：单选和多选均可；
    • d.填写信息：开发时限最大一个月，结束时间>开始时间>当前时间；
    • e.提交计划：保存。
      (3)修改
    • a.修改前提：操作人只允许对自己制定的计划进行修改，且须在计划开始前、计划执行人也未制定计划日程时，方可修改；
    • b.修改操作与新增计划操作相同。
