维护计划管理
      (1)查询
    • a.查询条件：除了“维护时限”，其他都可以选择为空，最大查询范围为三个月；
    • b.查询结果：显示计划时限包含在查询时间范围内的所有计划。
      (2)查看
    • a.双击查询结果列表下的一条记录，或选择一条记录，点击“查看”按钮；
    • b.弹窗显示该计划详情。
      (3)新增
    • a.点击新增：弹出新增维护计划页面；
    • b.查询客户：客户创建时间默认一个月，最大查询范围为一年；
    • c.选择客户：单选和多选均；
    • d.填写信息：维护时限最大为一个月，结束时间>开始时间>当前时间；
    • e.提交计划：保存。
      （4）修改
    • a.修改前提：操作人只允许对自己制定的计划进行修改，且在计划开始前、计划执行人也未制定计划日程时，可修改；
    • b.修改操作与新增计划操作相同。
      （5）删除
    • a.删除前提：与修改计划前提相同；
    • b.点击“删除”，确认即可。
