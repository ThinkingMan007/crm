积分兑换管理
      (1)查询积分兑换
    • 通过“发放状态”字段，可以看到当前礼品兑换信息的处理状态。
      (2)积分兑换
    • a.用户点击“积分兑换”，可以打开积分兑换弹窗；
    • b.在积分兑换弹窗中，输入联系人编码并点击“查询”按钮，可查询出该联系人的积分信息；
    • c.点击礼品名称后面的放大镜图标，可以打开选择礼品弹窗，进行礼品选择；
    • d.选择礼品并输入兑换数量后，点击“添加清单”，可将当前联系人兑换的礼品信息添加到下方列表中；通过变换联系人和礼品，可以向下方列表中添加多条礼品兑换信息；
    • e.选择下方列表中的兑换信息，并点击“移除清单”，可删除下方列表中的礼品兑换信息；
    • f.点击“暂存清单”，可以保存下方列表中的礼品兑换信息，同一营业部用户在下次点击积分兑换时，会自动加载上次保存的礼品兑换信息；
    • g.点击“生成清单”，下方列表中的礼品兑换申请会正式提交并生效，进入待审核状态；
    • h.点击“取消”，可以取消弹窗中的兑换信息并关闭积分兑换弹窗。
      (3)确认发放
    • 对于审批通过的积分兑换信息，营业部发放礼品后，可选择该条兑换信息，并点击“确认发放”，来设置该条兑换信息已经处理完成。
      (4)恢复库存
    • 对于状态为“营业部已发放”，但实际未发放礼品的积分兑换信息，可选择该条信息并点击“恢复库存”，来设置积分兑换信息仍处于礼品未发放状态。
