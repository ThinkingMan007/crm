角色管理

    依据用户需求不同，可以创建不同的角色例如营业员、分单员、组织架构管理员等等，以分单员为例，查看分单员功能权限。

      (1)新增角色(即:超级管理员对新增需求角色进行新增，并且分配功能权限：)
  1. a.点击“角色管理”弹出界面；
  2. b.点击“新增”按钮，出现创建界面；
  3. c.输入所要添加的角色名称和角色描述；
  4. d.双击综合管理系统，出现功能菜单，然后选择新角色所需要的功能权限；
  5. e.然后保存完成。
      6. (2)查询角色权限(即:对于刚刚创建的新角色或者已经存在的角色进行查询)
  6. a.在角色管理界面输入要查询的内容：统计员，点击“查询”；
  7. b.出现该创建角色，双击显角色属性；
  8. c.对该角色进行重新修改。
