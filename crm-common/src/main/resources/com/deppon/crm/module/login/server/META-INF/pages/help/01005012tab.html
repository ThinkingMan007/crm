发货周期表
      (1)查询
    • a.查询条件：选择已有分组，则查询该组内客户；不选择分组，则查询出本部门所有固定客户客户发货情况；
    • b.查询结果：按客户的发货量由高到底分页显示；
    • c.附加信息：双击查询结果列表下的一条记录，可以查看该客户的回访记录和线路分析情况。
      (2)制定计划
    • a.选择客户：在查询结果列表下选择一个或多个客户，点击“制定计划”按钮；
    • b.选择联系人：可选择一个或多个联系人；
    • c.填写维护信息；
    • d.保存：点击“制定计划”。
      (3)制定日程
    • a.选择客户：在查询结果列表下选择一个或多个客户，点击“制定日程”按钮；
    • b.选择联系人：双击联系人信息，在弹出的时间表中选择日期，点击“保存”。
