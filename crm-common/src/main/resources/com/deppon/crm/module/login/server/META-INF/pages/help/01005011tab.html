客户分组管理
      (1)新增分组
    • a.新增：新建一个分组，分组名不能与已有组名相同；
    • b.查看：双击一条分组记录，可查看该分组详情。
      (2)修改删除
    • a.修改：修改后新组名不能与已有组名相同；
    • b.删除：删除一个分组后，该组内的客户被释放出来，变成未分组客户。
      (3)分组管理
    • a.查询客户：选择已有分组，则查询出该组内客户；不选择分组，则查询出未分组客户；
    • b.选择客户：通过上移和下移筛选客户；
    • c.分配维护人：初次为该组选择维护人时，“分配维护人”为空，需选择该分组的维护人；
    • d.保存分组：点击“保存”按钮，提交分组信息。保存分组：点击“保存”按钮，提交分组信息。
