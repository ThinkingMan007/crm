用户管理

    该功能属于管理员权限，通过此功能菜单可以对CRM系统的功能权限进行管理，可以通过启动、禁用加载对系统的功能进行调整:

      (1)查询用户角色(即:首先对用户进行功能启用查询)
    • a.输入用户工号
    • b.选择是否启用，点击“查询”。当该用户功能未启用时，只能选择未启用选项才能进行查询；反之则选择启用进行查询
    • c.选择未启用的用户进行查询，双击用户详细信息列表中的未已启用用户，可以查看到该用户尚未添加角色
    • d.选择已启用的用户进行查询，双击用户详细信息列表中的已启用用户，可以查看出该用户的角色类型。
      (2)新增用户(新增用户，首先必须进行查询该用户是否已启用，如果属于已启用用户，则不能进行新增。)
    • a.进行用户是否启动查询；
    • b.对未启用的用户进行新增在新增界面选择未启用选项；
    • c.再带分配角色中进行角色选择；
    • d.选择所需要的角色就会出现在已分配中；
    • e.进行保存即可。
      (3)修改
    • 对已启用用户进行角色修改。
      (4)初始化密码(对全部用户密码进行设定或者修改。)
    • a.查询所要进行密码设置的用户；
    • b.点击初始化密码，弹出密码界面；
    • c.输入密码进行保存即可。
