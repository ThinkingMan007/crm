待办工单

    待办工单列表只显示需当前用户处理的工单，不同用户处理不同工单的操作分别如下：

      (1)工单审批申请经理待办，经理可操作：
    • a.点击“退回提交人”，录入退回原因，点击“提交”，工单则流入到提交人待办;
    • b.点击“完成处理”，工单流入任务部门工单管理。
      (2)投诉升级已升级公关部待办，公关部可操作：
    • a.点击“退回提交人”，录入退回原因，提交，工单则流入到提交人待办;
    • b.点击“完成处理”，显示界面比提交审批界面多了“最终反馈”，勾选“最终反馈”，只需填写反馈结果，提交后工单处理完毕。
      (3)工单审批退回处理人待办，可操作：退回上报人、提交审批、投诉升级、处理完成;
      (4)升级工单退回处理人待办，可操作：退回上报人、提交审批、投诉升级、处理完成;
      (5)任务对象退回处理人待办，可操作：退回上报人、提交审批、投诉升级、处理完成;
      (6)经理处理的任务对象退回经理待办，可操作：退回上报人、投诉升级、处理完成;
      (7)公关部处理的任务对象退回公关部待办，可操作：处理完成;
      (8)经任务对象申请延时处理人，可操作：延时审批，弹出框内按钮为退回提交人、同意：
    • a.点击“退回提交”，录入退回原因，提交，工单无法再申请延时;
    • b.点击“同意”，工单处理时间延时。
