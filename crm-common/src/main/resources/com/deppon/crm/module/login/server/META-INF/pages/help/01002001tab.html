潜客管理
      (1)查询潜客信息
    • 查询时，默认且只能查询本部门的潜客信息；创建时间的最大查询范围为一年。
      (2)新增潜客信息
    • a.点击“新增”按钮，弹出潜客编辑界面;
    • b.按要求填写客户信息;
    • c.信息填写完整后，点击“保存”按钮;
    • d.若系统已存在此客户信息，则提示“该客户已存在，请核实！”;
    • e.若系统已存在此固定客户信息，则提示“存在该联系方式的固定客户，不能新增或者修改固定客户！”。
      (3)修改潜客信息
    • a.查询：输入查询条件，显示符合查询条件的结果;
    • b.选择客户：选中要修改的潜客，点击“修改”按钮，进入潜客编辑界面;
    • c.修改信息：潜客信息中客户名称、联系人姓名、联系人手机、联系人电话不可修改，其他信息都可修改;
    • d.保存：修改完成后，点击“保存”按钮，信息保存成功；修改后点击“取消”按钮，信息不变。
      (4)下载“批量导入”模板
    • 点击“下载模板”按钮，模板下载成功。点击打开PotentialCust文档，即为潜客导入模板。用户可在文档中录入相关信息，以便“批量导入”操作。
      (5)批量导入
    • a.点击“批量导入”按钮，系统弹出温馨提示“请上传潜客附件”;
    • b.点击“浏览”按钮，弹出文件选择框，选择本地文件，文件格式必须是“下载模板”中下载下来的文档;
    • c.点击“上传”按钮，文件中的数据导入到潜客信息中;
    • d.“保存”完成后，对话框提示“导入成功潜客信息XX条，导入失败XX条，系统已存在未保存XX条”。
      (6)查看详情
    • a.查询：输入查询条件，点击“查询”，显示符合查询条件的结果;
    • b.查看：选中要查看的潜客，点击“查看详情”按钮，或者双击该潜客记录，弹出详情展示框，框内数据只可查看，不可编辑。
