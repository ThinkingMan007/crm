合同管理
      (1)新增合同
    • a.单击“新增”按钮，弹出合同编辑界面;
    • b.在“客户名称”中，查询签订合同客户;
    • c.填写合同基本信息（合同信息中的部门、合同状态不可填写）;
    • d.填写完成，点击“提交”按钮，保存。
      (2)修改合同同
    • a.填写查询条件，“创建时间”的最大查询范围为一年且必填，点击“查询”按钮，显示对应的查询结果;
    • b.选中需要修改的合同，点击“修改”按钮，弹出合同编辑界面;
    • c.客户基本信息不可修改，只能修改合同信息;
    • d.修改完成，点击“提交”保存。
      (3)查看合同
    • a.填写查询条件，“创建时间”的最大查询范围为一年且必填，点击“查询”按钮，显示对应的查询结果;
    • b.选中需要查看的合同，点击“查看详情”按钮，弹出合同信息界面;
    • c.点击“其他信息”按钮，查看合同的绑定部门、工作流信息等相关信息。
      (4)作废合同
    • a.填写查询条件，“创建时间”的最大查询范围为一年且必填，点击“查询”按钮，显示对应的查询结果;
    • b.选中需要作废的合同，点击“作废”按钮，弹出框选择“是”则生成作废申请工作流，选择“否”则取消作废。
      (5)绑定合同
    • a.填写查询条件，“创建时间”的最大查询范围为一年且必填，点击“查询”按钮，显示对应的查询结果;
    • b.选中需要作废的合同，点击“作废”按钮，弹出框选择“是”则生成作废申请工作流，选择“否”则取消作废;
    • c.单击“上传附件”按钮，选择相应附件;
    • d.单击“提交”按钮，生成绑定申请工作流。
      (6)解除绑定
    • a.填写查询条件，“创建时间”的最大查询范围为一年且必填，点击“查询”按钮，显示对应的查询结果;
    • b.选中需要解除绑定的合同，点击“解除绑定”按钮，弹出框选择“是”则解除绑定，选择“否”则取消解除绑定。
      (7)归属变更
    • a.填写查询条件，“创建时间”的最大查询范围为一年且必填，点击“查询”按钮，显示对应的查询结果;
    • b.选中需要变更归属部门的合同，点击“归属变更”按钮，弹出归属变更界面;
    • c.点击“上传附件”，选择要上传的附件;
    • d.点击“提交”按钮则生成归属变更工作流，点击“取消”则退出。
